SOI ISLP Website (Flask Web Application)

The SOI ISLP Website is a Flask-based application that allows Republic Polytechnic students from School of Infocomm (SOI) to apply for International Service-Learning Program (ISLP) trips, while staff can review, approve, and manage applications. It integrates Amazon RDS (MySQL) for data storage, AWS S3 for file management, and Amazon Cognito for authentication. Students can submit applications, view trip details, and receive email confirmations, while staff can filter applications, approve/decline submissions, and manage trip details.

Circlefinder.io (Flask Web Application)

The Circlefinder Website is a Flask application developed for an internship project (Straits Construction) to help the Quality and Service Department (QSD) monitor employee clock-ins at project sites. It features a Circle Generator for drawing circle, defining a radius, and generating editable circles on a map, with the option to download the results. This helps the company optimize network planning or surveillance camera placement by generating and visualizing coverage areas on a map to ensure comprehensive monitoring and connectivity. Additionally, the Upload Excel function reads Clock Records from Excel sheets, visualizes employee locations on a map, and allows filtering by employee and date, with adjustable radius settings for project sites. The tool enhances site monitoring with real-time data updates and interactive map-based visualizations.

Profile Pictures Editor (Flask Web Application)

The Profile Pictures Editor is a Flask application developed for an internship project (Straits Construction) to simplify the process of cropping employee profile photos. It allows users to drag and drop photos, select desired dimensions, and batch process multiple images efficiently. Using face detection technology, it automatically detects and crops faces while maintaining proper proportions to avoid distortion. Photos that don't meet the cropping requirements are placed in an "unacceptable" folder, with details logged in a text file. A progress bar provides real-time updates, and users can download the cropped photos as a zip file once processing is complete.
